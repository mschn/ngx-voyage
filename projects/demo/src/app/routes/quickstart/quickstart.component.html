Quickstart

Assuming you already have an Angular app, here's how you can install ngx-voyage.
If you want a more complete guide that includes setting up the Angular app from scratch, head to the Howto guide.

Install ngx-voyage and it's peer dependencies with npm:

ngx-voyage has peer dependencies on the following libraries, that will need to be installed as well:

  • Angular version ^19.2.0
  • PrimeNG version ^19.0.0, used as UI widget toolkit
  • highlight.js version ^11.11.1, used to syntax highlight code files
